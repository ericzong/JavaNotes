插件版本过低

有时，我们安装的插件版本过低，在高版本的 Eclipse 上将不能使用。这需要额外安装一个兼容插件，以Eclipse 4.6 为例：

  1. 菜单“帮助”-->“安装新软件”
  2. 对话框选择“http://download.eclipse.org/eclipse/updates/4.6”
  3. 加载工具列表中选择“Eclipse Tests, Tools, Examples, and Extras”-->“Eclipse 2.0 Style Plugin Suport”

合并两个版本的 Eclipse

Eclipse 发布一个大版本，都会按不同的开发模式分成多个独立工具包，比如 JEE 或 RCP。

如果想把几个独立工具包合并为一个，则需要进行一些文件合并，列举如下：

合并 plugins 和 features 目录。

合并配置文件：

  1. configuration\org.eclipse.equinox.source\source.info
  2. configuration\org.eclipse.equinox.simpleconfigurator\bundles.info
  3. configuration\org.eclipse.update\platform.xml
  4. artifacts.xml：合并 <artifacts> 所有子元素，并累加 size 属性。
