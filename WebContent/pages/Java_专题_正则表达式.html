概述

正则表达式（Regular Expressions）
描述一组具有共同模式的字符串的模式。
量化器
对其之前的内容进行量化，如：?、*、+ 等 。

核心 API：

  • Pattern 描述字符串模式
  • Matcher 测试一个字符，查看是否与模式匹配
  • PatternSyntaxException 定义的模式语法错误

API 使用


Pattern pattern = Pattern.compile("[Aa].*string");
Matcher matcher = pattern.matcher("A string");
boolean didMatch = matcher.matches();
int patternStartIndex = matcher.start();
int patternEndIndex = matcher.end();

模式匹配

常用 Matcher 方法：

  • matches() 整个输入序列是否与模式准确匹配
  • lookingAt() 搜索指定模式的子字符串
  • find() 搜索下一个匹配的子字符串
  • start() 匹配字符串在输入字符串中的起点索引
  • end() 匹配字符串在输入字符串中的终点索引

替换字符串

常用 Matcher 方法：

  • replaceAll() 将所有匹配值替换为一个指定的字符串
  • replaceFirst() 仅将首个匹配值替换为一个指定的字符串

分组

可以在模式中通过圆括号来创建分组。分组从左向右编号，从 1 开始（分组 0 表示完整的匹配结果）。

在替换方法中，指定字符串可以使用“$index”来引用分组。

常用 Matcher 方法：

  • group() 返回模式匹配的输入字符串，等价于 group(0)
  • group(int index) 返回模式匹配的分组
